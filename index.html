How the program works:

  1. A random quadratic (some a, b, and c value) is selected as the target.
  2. A dataset is created by evaluating the target quadratic for many random values of x, and then adding random noise to each y value.
  3. A random swarm of particles is initialized. Each particle is drawn to both its personal best found position and the best found position of the entire swarm.
  4. The graph on the left shows the swarm's best found position at each time step.
